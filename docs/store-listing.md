# Store listing

## Name

Weathercast — Rain Nowcast

## Subtitle

Know when rain reaches you

## Short description

Rain timing, intensity, duration, and confidence for the next 120 minutes.

## Description

Weathercast removes the clutter from weather. Choose a place or use foreground location to see whether rain is approaching, its likely arrival window, expected intensity, duration, and confidence.

Coverage is shown honestly. Precision regions can use licensed radar and dense observations; other regions fall back to enhanced or standard guidance. Forecast freshness and source are visible, saved places stay on your device, and no account or background tracking is required.

Weathercast is not an emergency-warning or safety-critical service. Follow official local alerts.

## Screenshot sequence

1. Now: clear or incoming rain headline
2. Timeline: intensity over 120 minutes
3. Radar: licensed live layer or honest regional fallback
4. Places: current and saved locations
5. Accuracy: transparent measurement definitions

Store metadata for Apple is mirrored in `store.config.json`. Final screenshots must be captured from signed production builds using live production data and current store-required device sizes.
